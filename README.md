# AI Fitness Trainer

An end-to-end Voicechat AI-powered fitness trainer that generates personalized workout plans and guidance based on user inputs and goals.

## üöÄ Features

- **Voice Interaction**: Have a natural conversation with an AI assistant to describe your fitness goals, experience, and limitations.
- **Personalized Plans**: Generates structured workout schedules and diet plans tailored to your specific needs.
- **Interactive UI**: View your generated plans in a clean, responsive dashboard.
- **User Authentication**: Secure sign-up and login via Clerk.
- **Real-time Updates**: Plans are generated and saved instantly to the database.

- **Webhooks**: [Svix](https://www.svix.com/) (for Clerk webhooks).

## üèóÔ∏è Architecture

1.  **Voice Input**: The user interacts with the Vapi voice assistant on the frontend.
2.  **Data Collection**: Vapi collects user details (age, weight, goal, etc.) during the conversation.
3.  **Plan Generation**: Vapi triggers a webhook to the Convex backend (`/vapi/generate-program`).
4.  **AI Processing**: The Convex function uses Google Gemini to generate a JSON-structured workout and diet plan.
5.  **Data Storage**: The generated plan is validated and stored in the Convex database.
6.  **UI Update**: The frontend (using Convex React hooks) automatically updates to show the new plan.

## ‚öôÔ∏è Installation & Setup

1.  **Clone the repository**:
    ```bash
    git clone <repository-url>
    cd ai-fitness-trainer
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Environment Variables**:
    Create a `.env.local` file in the root directory and add the following keys:

    ```env
    # Convex
    CONVEX_DEPLOYMENT=... # Generated by npx convex dev
    NEXT_PUBLIC_CONVEX_URL=...

    # Clerk
    NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=...
    CLERK_SECRET_KEY=...
    CLERK_WEBHOOK_SECRET=...

    # Vapi
    NEXT_PUBLIC_VAPI_API_KEY=...
    NEXT_PUBLIC_VAPI_WORKFLOW_ID=...

    # Google Gemini
    GEMINI_API_KEY=...
    ```
    *Note: `GEMINI_API_KEY` and `CLERK_WEBHOOK_SECRET` are used in the Convex backend functions. Ensure they are set in your Convex dashboard or environment configuration.*

4.  **Start the Convex backend**:
    ```bash
    npx convex dev
    ```

5.  **Start the Next.js development server**:
    ```bash
    npm run dev
    ```

6.  **Open the app**:
    Visit [http://localhost:3000](http://localhost:3000) in your browser.

## üí° Usage

1.  **Sign Up/Login**: Create an account using Clerk.
2.  **Start Conversation**: Go to the "Generate Program" page and click "Start Call".
3.  **Provide Details**: Speak with the AI assistant about your fitness stats and goals.
4.  **View Plan**: Once the call ends, you will be redirected to your profile to view your personalized workout and diet plan.
